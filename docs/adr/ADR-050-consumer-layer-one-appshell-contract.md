# ADR 050 The Consumer & Fan Experiences layer (session 1): one AppShell contract, an audit that corrects the prompt's stale premises, two deliberate deviations

Status: accepted 2026-07-13. Context: this is session 1 of the Consumer & Fan Experiences layer, the first structural work unblocked by ADR-049 (reference-is-intent, CONSISTENCY criterion). Only `apps/web` was touched — `apps/api`, `packages/`, `programs/` are untouched, so no service, engine, bus, or chain behaviour moves (architecture law upheld). The originating prompt carried several stale premises (two shells, recharts present, flags/depth/halt/⌘K missing, token debt), all of which prior sessions had already resolved; an import-graph audit of 118 web files this session verified the real state and is recorded here so future sessions do not redo the work.

Decision:
- **One AppShell contract.** Extracted a single `apps/web/src/components/ui/AppShell.tsx` — `{ticker?} + TerminalHeader + children + Footer` on the standard `flex min-h-screen flex-col bg-bg` frame — as the ADR-049 CONSISTENCY primitive. Every route SHOULD render through it. `/moments` and `/replay` are migrated; the ~9 remaining feature pages (Home `HomeShell` wrapper, terminal, leaderboard, portfolio, `moments/[id]`, bracket, how-it-works, history, settings, profile) still self-assemble `TerminalHeader + Footer` inline and are queued to route through AppShell next. Two intentional shell exceptions: `/onboarding` (own minimal `Wordmark` header, no nav) and `/proofs` (`StubScreen`).
- **Audit — corrections of record (already true before this session):** (1) the "two shells" are ONE — the losing App/Moments shell (`features/home/Header.tsx`, `components/ui/ScreenSwitcher.tsx` App|Terminal toggle, `features/northstar/NorthStarScreen.tsx`) was already deleted from disk; `TerminalHeader` is the sole chrome. (2) `recharts` is already gone (ADR-048): `PortfolioCard` uses the `EquityCurve` SVG, `AttackMomentum` is hand-rolled SVG, and it is absent from `package.json`. (3) Real HD flags already exist (`components/ui/Flag.tsx` → flagcdn with graceful fallback). (4) Depth tabs already exist (`MatchTabs` + `terminal/depth/`). (5) A Framer-Motion amber `HaltBanner` already exists. (6) A `cmdk` ⌘K `CommandMenu` already exists. (7) Token-law is clean — zero raw-hex arbitrary colors across 118 files; the 185 bracket-value Tailwind utilities are legitimate (`tracking-[…]`, `max-w-[…]`, `shadow-[var(--up)…]`, Radix `data-[state]` selectors), not debt.
- **Dead code (2 true orphans deleted):** `features/home/CompetitionBar.tsx` (remnant of the deleted App board) and `components/ui/BoothBubble.tsx` (a `return null` placeholder stub). **DEVIATION from the prompt's "delete everything imported by nothing":** `lib/api.ts` and `lib/ws.ts` are KEPT — they are the documented integration seams the global-track wiring (ADR-042/048) builds the typed client + WS hook on; deleting next-phase scaffolding would be counterproductive.
- **/moments hero (3.5):** replaced the flat uniform grid (the anti-reference named in CLAUDE.md) with a Moment-of-the-Day hero (`components/ui/MomentHero.tsx`) — large, the biggest swing, with a Framer-Motion River path draw-on for delight (reduced-motion honored, tokens only, inline SVG per ADR-045) — and demoted the rest to a "More moments" grid. Screenshotted and visually verified at lg + xl (mobile out of scope this run).

Consequences: the consumer layer now has one shell to converge on, and `/moments` leads with a hero instead of a grid. REMAINING (queued, not done this session): route the ~9 pages + `/proofs` through AppShell; 3.1 bigger crests/team-colors (blocked — player crests need an API-Football key, unavailable); 3.2 elevate the halt into the full River-integrated money-shot; 3.3 build a real live Booth motion ticker (the `BoothBubble` stub was deleted, so build fresh); 3.4 fill depth tabs from TxLINE/API-Football; 3.6 polish audit (skeletons, empty states, ⌘K verification). VERIFY: `AppShell.tsx` exists and `/moments` + `/replay` render through it; `CompetitionBar.tsx` and `BoothBubble.tsx` are gone; `lib/api.ts` + `lib/ws.ts` still present; `grep recharts apps/web/package.json` empty; zero raw hex in components; `MomentHero` present and its motion honors `prefers-reduced-motion`. Cross-ref: ADR-049 (reference-is-intent + design-cop 9–11 + primitives-default — the keystone this builds on), ADR-043 (original shell/Home lock), ADR-045 (inline-SVG viz precedent), ADR-032/041 (Moments), ADR-021 (replay plane), ADR-048 (the fixture-wired deploy this elevates).
