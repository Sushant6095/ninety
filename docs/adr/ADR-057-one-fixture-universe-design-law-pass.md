# ADR 057 One fixture universe + the 2026-07-14 design-law pass

Status: accepted 2026-07-14 (`merge/live-integration`). Cross-ref: ADR-051 (two-source rule — this is its web-fixture enforcement), ADR-052 (animation libs), ADR-054 (matchLiveStore SSOT), ADR-055 (baked-asset hex exemption), ADR-049 (feeling criterion), ADR-056 (landing owns `/`).

Context: hand-written parallel slates (TICKER, TERM_MARKETS) drifted from MARKETS — Argentina in two matches at once, the same stage named both R16 and R32, CAN–MAR simultaneously live and upcoming. Design-cop FAILed the session on arbitrary letter-spacing and sub-44px targets; the stats-bar and lineup dots tinted teams with price-direction tokens (rail showed EGY green because its price was up while the stats header colored AUS green 200px away).

Decision:
- ONE fixture universe (the A2 law): `apps/web/src/lib/fixtures.ts` MARKETS is the single fixture source for the whole app; TICKER and TERM_MARKETS (`lib/terminal.ts`) are DERIVED from it — hand-written second slates are forbidden. AUS–EGY joined MARKETS so board/rail/ticker/terminal literally read one matchLiveStore entry. R16 = exactly 16 matches / 32 unique teams; R32 is the settled past (proofs page carries receipts). openPrices seed from the trace's first point (spark[0]) so every Δ-vs-open is real, never 0.0 forever. Positions/fills/moments/proofs aligned to the one universe (ENG–AUS and PAR–FRA ghosts removed; terminal and /portfolio positions agree share-for-share). Acceptance gate: the read-out-loud test — enumerate every text element on a still; if any two disagree, it is not done.
- up/down are price-direction tokens ONLY — never team identity. Side identity is positional/labeled; color carries emphasis only (leading side bg-hi/60, trailing bg-hi/20, mirroring hi/lo value text). W/D/L form chips remain legitimate status use (labeled letters, never color alone).
- Tracking is a token scale + 44px hit targets are law: `tokens.ts` exports `tracking` (micro 0.08em / tag 0.1 / label 0.12 / caps 0.14 / banner 0.16 / hero 0.2 / wide pinned 0.025 / tight −0.02 / tighter −0.03) mapped through tailwind.config letterSpacing; zero arbitrary tracking-[…em] in src. Stacked rows use real min-h-11; visually-compact horizontal controls use the `.hit` utility (globals.css — 44px centered invisible strip). These were the two design-cop FAIL drivers; verdict flipped to SHIP (`design/verdicts/2026-07-14-session-finals.md`).
- Registry pulls may be REJECTED and must be ledgered: 21st.dev #17170 Astryx Avatar Group refused on read — it re-exports Meta's `@astryxdesign/core` theme package, a second design system (banned by library law). Quota spend + rejection reason ledgered in `design/PROVENANCE.md`. CrowdCall (H2H who-wins vote, sofascore vote-and-poll.md) shipped from 21st.dev #8243 Review Filter Bars — structure kept, re-skinned to tokens.
- The landing's conversion element is THE terminal button: "Open the terminal" filled CTA verbatim in header, hero, and close (hyperfoundation repetition law, ADR-056); credits CTA is the outline secondary. Site metadata baseline: `public/favicon.svg` (brand 90 mark; hex inlined in the asset — same exemption as baked flag PNGs, ADR-055), `public/og.png` (fresh 1200×630 board capture mid-halt), openGraph + twitter summary_large_image, metadataBase https://ninety-nu.vercel.app.
- MotionScore posture: the previous S was a cheap S (zero animations detected). The current B grades 20 real animations — Animations A, GPU Pressure S, Scroll A, Thrashing C. Every flagged thrash cause fixed same-session: gsap from() → fromTo() (explicit values, no computed-style read at start) and will-change promoted only while a tween runs (set on start, clearProps on complete — pre-promoting below-fold sections would spend the GPU texture budget). Fresh re-grade blocked by MotionScore free-tier caching; fixes are deployed.

Consequences / VERIFY: any second slate, raw tracking-[…], sub-44px interactive target, or team-tinted up/down is a bug. LAW (owner applies — CLAUDE.md edits are the user's call): design-law section should add "ONE fixture universe: web fixtures come only from `lib/fixtures.ts` MARKETS (derived slates fine, hand-written duplicates are bugs). up/down tint price direction only, never team identity. Letter-spacing only via the `tracking` token scale; interactive targets ≥44px (min-h-11 or .hit)."
