# ADR 069 The long landing — a football-first scroll story below the kept notio hero

Status: accepted 2026-07-17 (`landing/long-page` → `merge/live-integration`). Builds on ADR-066/068 (the notio landing + hero) and ADR-058/052 (dynamism + GSAP). Preflight was declared and approved before any code; the 3D "wow" was explicitly deferred to a separate run (this keeps a clean pre-3D MotionScore baseline).

Context: the notio landing was a strong hero followed by three thin generic sections (features/stats/footer). The ask: a long, dense, football-first scroll story (Hyperliquid/hyperfoundation density) that brings back the globe, the particle count-up, and the proof section, fills the voids, and pitches the Consumer track — WITHOUT touching the 10/10 hero and WITHOUT any new 3D this run.

Decision — **compose, don't rebuild: reuse the already-built, MotionScore-tuned landing pieces under the kept notio hero, add two new consumer sections, drive it all with the existing GSAP scroll spine.**
- New `features/landing/LandingLong.tsx` composes: **1** notio Navbar + **2** notio Hero (kept, HeroRiver) → `LandingScroll` wrapping **3** THE LOOP (`LoopStage`, real `useHaltSequence` on scroll-in) · **4** PRICE IS PROBABILITY (`PriceVoid` + `PriceScrub`, the giant 61.4) · VelocityBand chapter break · **5** THE FOOTBALL EXPERIENCE (new) · **6** PROOF/ON-CHAIN (`FlowField` + `Highlighter` + steps) · the Booth · **7** THE WHOLE TOURNAMENT (`NumberTicker` 104/48/1,000 + `WorldGlobe`) · FreeCredits · **8** GAMES+MOMENTS+TELEGRAM (new) → **9** the notio "opening bell" footer + CrowdBand. `app/page.tsx` renders `<LandingLong/>`.
- **Two new sections only:** `FootballExperience.tsx` (the 5-step fan journey — the Consumer-track pitch, teasers so no live game mounts) and `ConsumerBento.tsx` (magicui BentoGrid/BentoCard + Marquee, already-pulled, re-skinned — Next Goal / Moments / Telegram cards). Everything else is reuse.
- **Motion is a ceiling, not a target (per the amendment):** ~10 already-tuned components carry the page; the two new sections add restraint. **Cut** the skiper52 pillars (animated `flex-grow` = layout thrash) and the duplicate old close (the notio footer IS the "opening bell").
- **Read-out-loud fix:** `BoothQuotes` was hardcoded to Australia–Egypt while the hero/loop/price are all the featured **CAN-MAR** (`FEATURED = wc26-can-mar`); re-homed it to inline CAN–MAR demo commentary so every surface reads one match.

Rejected: (a) a new 3D scene this run — deferred by the user to keep a clean MotionScore baseline; (b) rebuilding sections from scratch — the old `LandingPage` already solved them and passed MotionScore; (c) rewiring the `LoopStage`/`useHaltSequence` reprice — out of scope for a composition task (see the flag).
